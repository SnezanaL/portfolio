Project details

For this project, I integrated with the REST Countries API to create a dynamic and interactive web application that allows users to explore information about countries. The application is built using Angular 8, Material Angular and RxJS on the front-end, providing a seamless and responsive user experience.

In addition to the technologies mentioned earlier, this project leverages Firebase Authentication to provide seamless and secure user authentication. The integration allows users to log in using their existing Facebook or Google+ accounts, streamlining the onboarding process. Firebase Authentication provides a robust and scalable solution, handling user authentication, identity verification, and access control.

Technologies Used:

  1. Angular 8 (Web App) Utilized for building the interactive and responsive web application, offering a seamless learning experience on various devices.
  2. Angular Material is a UI component library designed for Angular applications, following the Material Design principles. It provides a set of pre-built and customizable components such as buttons, cards, forms, and more. Using Angular Material ensures a consistent and visually appealing user interface in your Angular applications.
  3. RxJS: Utilized to handle asynchronous operations and events, enhancing the efficiency and responsiveness of the application.
  4. Sass: Implemented for styling the user interface, allowing for maintainable and organized CSS.
  5. Firebase Authentication: Firebase Authentication is utilized to handle user sign-up, sign-in, and account management seamlessly.
  6. Social Login Options: Users have the convenience of logging in using their Facebook or Google+ credentials. This feature enhances user engagement and eliminates the need for creating additional accounts.
  7. Single Sign-On (SSO): The integration of social login options contributes to a Single Sign-On experience, allowing users to access the application with their preferred identity providers.

Key Features:

  1. Homepage Overview: Users can view a list of all countries fetched from the REST Countries API on the homepage.
  2. Search Functionality:Implemented a search feature enabling users to find specific countries quickly. The search is facilitated through an input field for a smooth user experience.
  3. Region Filtering: Countries can be filtered by region, allowing users to narrow down their exploration based on geographical areas.
  4. Detailed Country Information: Clicking on a country provides users with more detailed information on a separate page. This includes data such as population, area, and more.
  5. Border Country Navigation: Users can seamlessly navigate to bordering countries directly from the detailed country information page.
  6. Color Scheme Toggle: Implemented an optional feature allowing users to toggle between light and dark color schemes, enhancing the visual experience based on user preferences.
