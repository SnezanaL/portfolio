Quiz App

A simple quiz app built using React 18. The app fetches questions from a JSON file located in the public folder and displays multiple-choice questions to the user. The app tracks the user's score and displays it at the end of the quiz. The app uses React Context and Reducer hooks for efficient state management.

Key Features:

  1. Fetches questions from a JSON file located in the public folder.
  2. Displays multiple-choice questions to the user.
  3. Tracks the user's score and displays it at the end of the quiz.
  4. Uses React Context and Reducer hooks for efficient state management.
