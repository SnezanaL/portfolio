Project details

In this project, I was responsible for the frontend development of a desktop accounting software. The application was built using a combination of modern and robust technologies including Angular, Angular Material, RxJS, SASS, and Electron.

My responsibilities encompassed:

  1. Angular Development: I utilized Angular, a powerful and scalable framework, to build the user interface of the application. This involved creating dynamic and interactive components that responded to user inputs and events.
  2. Angular Material: I used Angular Material to design a modern and user-friendly interface. This library provided a wide range of pre-built components that adhered to Material Design principles, ensuring a consistent and appealing aesthetic across the application.
  3. Electron: I used Electron to package the application as a standalone desktop application. This allowed the application to be installed and run on Windows, MacOS, and Linux operating systems.
  4. Responsive Design: I ensured that the application was responsive and accessible across a variety of devices and screen sizes. This involved implementing a mobile-first approach and utilizing media queries to adapt the layout and design to different viewports.
  5. RxJS: I employed RxJS, a library for reactive programming, to handle asynchronous data calls and manage state within the application. This allowed for efficient and seamless data flow, improving the overall performance and user experience of the software.
  6. Adobe XD: I used Adobe XD to create wireframes and mockups of the application's user interface. This involved designing the layout, navigation, and visual elements to ensure a cohesive and intuitive user experience.

Throughout the project, I adhered to best coding practices, ensured code quality, and worked collaboratively with the team to meet project deadlines and deliver a high-quality product.
